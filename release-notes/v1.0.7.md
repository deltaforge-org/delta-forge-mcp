## What's Changed

The DeltaForge MCP server: the Model Context Protocol bridge that exposes DeltaForge SQL validation, formatting, lineage, documentation search, and catalog browsing to AI assistants.

### Aligned to engine 1.0.7

- The server version now tracks the DeltaForge engine release. The embedded documentation and SQL knowledge are rebuilt from the 1.0.7 corpus, so `search_docs`, `get_command_syntax`, and `validate_syntax` reflect the current command surface.
- Documentation coverage picks up the 1.0.7 engine additions, including the GPU-accelerated graph algorithms and the statistics-driven record-anchor auto-detection for JSON and XML in `DISCOVER`.
