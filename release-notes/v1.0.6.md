DeltaForge MCP Server 1.0.6.

MCP server that gives Claude, Cursor, Copilot, and any MCP client typed
access to your DeltaForge install: Delta Lake and Iceberg catalog, lineage,
SQL analysis, docs, and pipelines under the same RBAC and audit as a human.

Includes the server binary for Linux (.tar.gz), macOS (.tar.gz), and
Windows (.zip). The npm package 'deltaforge-mcp' downloads these assets on
install; run with 'npx deltaforge-mcp'. See README.md and client-config/ for
setup.

More at https://deltaforge.org/pages/mcp-server.html
